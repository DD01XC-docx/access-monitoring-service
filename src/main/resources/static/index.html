About Seerlogg

Seerlogg is an open source security analytics platform that unifies XDR and SIEM capabilities for modern security teams.

We built it to cut through noisy alerts and give analysts a clear view of threats across endpoints and cloud workloads.

With an API-first, self-hostable architecture, Seerlogg lets you extend detections, dashboards and workflows without vendor lock-in.

Open source MIT / Apache-2.0
Built for SOC & DevOps teams
Stack Java · Postgres · Docker
